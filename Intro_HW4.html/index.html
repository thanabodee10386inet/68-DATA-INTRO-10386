[ประวัติส่วนตัว] [ตารางรายวิชา]

ข้อมูลหลักสูตร

ชื่อหลักสูตร : สาขาวิศวกรรมสารสนเทศและเครือข่าย

รหัสหลักสูตร : 67062043

ระดับการศึกษา : ปริญญาตรีเทียบโอน 2 - 3 ปี

คณะ : เทคโนโลยีและการจัดการอุตสาหกรรม

สถานศึกษา : มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ

คณะเทคโนโลยีและการจัดการอุตสาหกรรมมุ่งเน้นสร้างบัณฑิตที่เชี่ยวชาญทั้งด้านเทคโนโลยีและการบริหารจัดการ เพื่อตอบรับภาคอุตสาหกรรมยุคใหม่ นักศึกษาจะได้เรียนรู้หลักการวิศวกรรม การผลิต การจัดการซัพพลายเชน การควบคุมคุณภาพ และการประยุกต์ใช้เทคโนโลยีดิจิทัล เช่น AI, IoT, Big Data เพื่อเพิ่มประสิทธิภาพและสร้างมูลค่าแก่องค์กร
